🚀 FAST SETUP (5 MINUTES):
============================

1️⃣  INSTALL XAMPP
   Download: https://www.apachefriends.org/
   Install default options
   
2️⃣  INSTALL MOSQUITTO
   Download: https://mosquitto.org/download/
   Install default options
   
3️⃣  CONFIGURE MOSQUITTO
   Edit: C:\Program Files\mosquitto\mosquitto.conf
   Delete all, then paste:
   
   listener 1883
   protocol mqtt
   
   listener 9001
   protocol websockets
   
   allow_anonymous true
   
   Save!

4️⃣  COPY PROJECT FILES
   Copy entire "notif" folder to: C:\xampp\htdocs\
   Result: C:\xampp\htdocs\notif\

5️⃣  START SERVICES
   Terminal 1:
   cd "C:\Program Files\mosquitto"
   mosquitto -v
   
   (Keep running - you should see "Opening websockets listen socket on port 9001")
   
   Then open XAMPP Control Panel:
   - Start Apache
   - Start MySQL

6️⃣  CREATE ACCOUNT & LOGIN
   Visit: http://localhost/notif/signup.html
   - Create new account
   - Fill in username, email, password
   - Click Sign Up
   - Login with credentials

7️⃣  OPEN DASHBOARD
   http://localhost/notif/
   
   Your notifications appear as BEAUTIFUL BADGES!

8️⃣  SEND TEST ALERT
   Go to: http://localhost/notif/admin.html
   - Select topic (School/System/Weather)
   - Type a message
   - Click "Send Alert"
   - See badge appear on dashboard immediately!

🔧 TROUBLESHOOTING:
===================

Problem: "Can't connect to MQTT"
Solution: Make sure Mosquitto.exe -v is running in terminal

Problem: "Database error"
Solution: 
1. Check MySQL is running in XAMPP
2. Go to http://localhost/phpmyadmin
3. Database created auto on first login

Problem: "Page shows blank"
Solution:
1. Check Apache is running in XAMPP
2. Files should be in C:\xampp\htdocs\notif\
3. Clear browser cache (Ctrl+Shift+Delete)

Problem: "No notifications appear"
Solution:
1. Mosquitto running? Check terminal shows "port 9001"
2. Browser console errors? Press F12
3. Logged in? Must be logged in to see notifications
4. Admin panel can send test alert

7. All connected users receive instantly
8. Badges appear on dashboards
