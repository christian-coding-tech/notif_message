📋 INSTALLATION STEPS:
======================

STEP 1: Download & Install Required Software
============================================
□ XAMPP (Apache + MySQL + PHP)
  Download: https://www.apachefriends.org/
  
□ Mosquitto MQTT Broker
  Download: https://mosquitto.org/download/

STEP 2: Configure Mosquitto
==========================
□ Edit: C:\Program Files\mosquitto\mosquitto.conf
□ Delete all content
□ Paste:
  listener 1883
  protocol mqtt
  
  listener 9001
  protocol websockets
  
  allow_anonymous true
□ Save

STEP 3: Copy Project
===================
□ Copy entire "notif" folder to: C:\xampp\htdocs\
□ Result: C:\xampp\htdocs\notif\

STEP 4: Start Services
=====================
□ Terminal 1: Run Mosquitto
  cd "C:\Program Files\mosquitto"
  mosquitto -v
  (Keep running - should see "Opening websockets listen socket on port 9001")

□ Terminal 2 (or XAMPP): Start Apache & MySQL
  Open XAMPP Control Panel
  Click Start for Apache
  Click Start for MySQL

STEP 5: Create Account
=====================
□ Visit: http://localhost/notif/signup.html
□ Create new account
□ Login

STEP 6: Use System
=================
□ Dashboard: http://localhost/notif/
□ Admin: http://localhost/notif/admin.html
